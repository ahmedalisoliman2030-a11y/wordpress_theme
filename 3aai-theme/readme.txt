A modern, dark-mode ready FSE theme designed for tech companies, startups, and SaaS projects. Optimized for speed and accessibility.

== Description ==

3AAi-Theme is a cutting-edge Full Site Editing (FSE) block theme specifically crafted for technology companies, startups, and SaaS businesses. With its modern dark-mode design, comprehensive block patterns, and performance-first approach, this theme helps you create stunning websites without writing a single line of code.

= Key Features =

* **Full Site Editing**: Complete control over your site's design using the WordPress block editor
* **Dark Mode Ready**: Beautiful dark color scheme optimized for tech companies
* **Block Patterns**: Pre-designed patterns including Hero sections, Services grids, Tech stack showcases, and Pricing tables
* **RTL Support**: Full support for right-to-left languages including Arabic
* **Performance Optimized**: Uses system fonts for lightning-fast load times
* **Accessibility Ready**: Built with accessibility best practices
* **Responsive Design**: Looks great on all devices and screen sizes
* **Customizable Colors**: Easy-to-customize color palette through theme.json
* **Modern Typography**: Clean, readable typography with flexible font sizes

= Block Patterns Included =

* **Hero Tech**: Modern hero section with gradient background and call-to-action buttons
* **Services Grid**: 3-column grid for showcasing your services or features
* **Tech Stack**: Logo showcase for displaying technologies or partners
* **Pricing Table**: Professional pricing table with featured plan highlight

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Type '3AAi-Theme' in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Editor to start customizing your site.

== Frequently Asked Questions ==

= Does this theme support Full Site Editing? =

Yes! This theme is built specifically for Full Site Editing (FSE) and requires WordPress 6.4 or higher.

= Can I use this theme in Arabic? =

Absolutely! The theme has full RTL (Right-to-Left) support and is translation-ready.

= How do I customize the colors? =

You can customize colors through the WordPress Site Editor (Appearance > Editor) or by editing the theme.json file.

= Is the theme compatible with page builders? =

This theme is designed for Full Site Editing and works best with the WordPress block editor. Page builders are not recommended.

= How do I add the block patterns to my pages? =

In the block editor, click the '+' button and look for the '3AAi Patterns' category. You can insert any pattern with a single click.
